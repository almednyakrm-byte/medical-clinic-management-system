# نظام إدارة عيادات طبية مع حجز مواعيد ومراقبة سجلات المرضى
==========================

## ملخص المشروع
----------------

نظام إدارة عيادات طبية مع حجز مواعيد ومراقبة سجلات المرضى هو نظام معلوماتي يهدف إلى تسهيل إدارة عيادات طبية من خلال حجز مواعيد وتتبع سجلات المرضى.

## هيكلية المشروع
-----------------

### مجلدات المشروع

* `app`: مجلد تطبيق النظام
* `docker`: مجلد تكوينات docker
* `docs`: مجلد الوثائق
* `tests`: مجلد الاختبارات

### تكوينات docker

* `docker-compose.yml`: تكوين docker-compose
* `docker-compose.override.yml`: تكوين docker-compose override

## تعليمات تشغيل النظام
-------------------------

### تشغيل النظام باستخدام docker-compose

1. تأكد من أن docker و docker-compose مثبتين على نظامك.
2. انتقل إلى مجلد المشروع.
3. تشغيل الأمر التالي:
bash
docker-compose up -d

4. انتظر حتى النظام يبدأ تشغيله.
5. افتح متصفح ويب ومرر إلى `http://localhost:8000` لفتح تطبيق النظام.

## وحدات النظام
-----------------

### وحدات النظام

* `appointments`: وحدات حجز مواعيد
* `patients`: وحدات مراقبة سجلات المرضى
* `doctors`: وحدات إدارة أطباء

### جداول قاعدة البيانات

* `appointments`: جدول حجز مواعيد
* `patients`: جدول مراقبة سجلات المرضى
* `doctors`: جدول إدارة أطباء

### أدوار النظام

* `admin`: دور إدارة النظام
* `doctor`: دور إدارة أطباء
* `patient`: دور مراقبة سجلات المرضى
